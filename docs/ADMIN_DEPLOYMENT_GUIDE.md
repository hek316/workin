## 👥 직원 온보딩 방법

### 1단계: 공지 전달

다음 3가지 방법 중 하나 이상을 사용하세요:

#### 방법 1: 카톡/메신저로 안내
```
[워크인 설치 안내]

📱 워크인 출퇴근 기록 시스템이 오픈되었습니다!

🔗 설치 링크
https://workin-vac1.vercel.app

📲 설치 방법
1. 위 링크를 누르세요
2. "설치 방법 보기" 버튼을 눌러 안내를 따라하세요
3. 홈 화면에 앱 아이콘이 생성됩니다

❓ 문제 발생 시
앱 내 "설치 도움말" 메뉴를 확인하거나
관리자에게 문의해주세요
```

#### 방법 2: 이메일로 안내
제목: [필독] 워크인 출퇴근 앱 설치 안내

본문 예시:
```
안녕하세요,

기존 수기 출퇴근 기록을 대체할 워크인 앱이 배포되었습니다.

■ 워크인이란?
GPS 기반으로 자동화된 출퇴근 기록을 제공하는 시스템입니다.

■ 설치 방법
1. 모바일 기기에서 아래 링크를 열어주세요
   https://workin-vac1.vercel.app

2. "설치 방법 보기"를 눌러 안내를 따라주세요
   - Android: Chrome 브라우저 사용
   - iPhone: Safari 브라우저 사용

3. 홈 화면에 앱을 추가하면 완료입니다

■ 첨부 파일
- QR 코드 (스마트폰 카메라로 스캔하세요)
- 상세 설치 가이드

■ 문의
설치 중 문제가 발생하면 관리자에게 연락주세요.

감사합니다.
```

#### 방법 3: 오프라인 포스터
1. `docs/install-poster.html`을 브라우저에서 열기
2. 인쇄 (Ctrl/Cmd + P)
3. 사무실 눈에 잘 띄는 곳에 부착

---

### 2단계: 설치 지원

#### 설치 시연 (추천)
직원들을 모아 5분 실시간 시연:

1. **QR 코드 스캔** (또는 URL 직접 입력)
2. **브라우저 확인** (Android: Chrome, iOS: Safari)
3. **메뉴 열기**
   - Android: 우측 상단 점 3개(⋮)
   - iOS: 하단 공유 버튼(⬆︎)
4. **"홈 화면에 추가"** 선택
5. **완료 확인** (홈 화면에 아이콘 생성)

#### 개별 지원
설치를 어려워하는 직원을 위해:
- 상세 가이드 URL 전달: https://workin-vac1.vercel.app/install
- 스크린샷과 함께 단계별 안내
- 직접 설치 도와주기

---

### 3단계: 설치 확인

#### 확인 방법
각 직원에게 다음을 요청:
1. 홈 화면 스크린샷 공유
2. 앱 실행 후 로그인 확인


## 🔧 문제 해결 가이드

### 자주 발생하는 문제

#### 1. "홈 화면에 추가" 메뉴가 안 보여요

**Android 사용자**
- ❌ Samsung 인터넷, Naver 앱 등 사용 중
- ✅ Chrome 브라우저로 접속하도록 안내

**iOS 사용자**
- ❌ Chrome, Kakao 앱 등 사용 중
- ✅ Safari 브라우저로 접속하도록 안내

#### 2. 설치했는데 홈 화면에 아이콘이 없어요

**해결 방법**
1. 홈 화면을 여러 페이지 확인 (좌우 스와이프)
2. 앱 서랍 또는 앱 목록 확인
3. 검색으로 "워크인" 찾기

#### 3. 설치는 되었는데 실행이 안 돼요

**해결 방법**
1. 인터넷 연결 확인 (Wi-Fi 또는 데이터)
2. 앱 삭제 후 재설치
3. 브라우저 캐시 삭제

#### 4. GPS 권한을 어떻게 주나요?

**Android**
1. 설정 → 앱 → Chrome → 권한 → 위치
2. "앱 사용 중에만 허용" 선택

**iOS**
1. 설정 → Safari → 위치 → 허용
2. 또는 앱 실행 시 권한 요청 팝업에서 "허용"

#### 5. 오래된 정보가 보여요 (업데이트 안 됨)

**해결 방법**
1. 앱을 완전히 종료
2. 다시 실행 (자동 업데이트됨)
3. 또는 브라우저에서 새로고침 (Ctrl/Cmd + Shift + R)
